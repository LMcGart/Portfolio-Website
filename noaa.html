Project Description

The client, GLANSIS, describes that the main issue with their current website is that visitors have a difficult time finding the proper resources to fulfill their goals on the site. Through the addition of familiar navigation, modern styling, and an overhaul to the format of GLANSIS’ current website, my redesign attempts to solve this problem and create an updated site with a logical flow of information and help wherever the user may need.

The Problem

The largest issue that people accessing GLANSIS face is not being able to access information with ease and without confusion. There are many tools offered by GLANSIS, with the main ones being the “Species List Generator,” “Map Explorer,” and “Risk Assessment Clearing House.” These tools hold a wealth of information but the website is currently organized in a way that is confusing for the average person to use effectively and efficiently. In addition, GLANSIS does not currently have an up-to-date mobile view. As seen in the mobile screenshot below, the information is not organized well and the form entries run into the margin. As shown in the desktop view, there is much going on and it is hard for the viewer to distinguish process it all.
